Keychron K2

Introduction

The Keychron K2 is a mechanical keyboard showcasing a compact 75% keyboard layout. It offers competitive features, many of which are customizable to the user's taste and needs. This documentation will provide you with all the information you need regarding this device and the variety of features and options it offers.

Details

Compatibility

The K2 is one of the few mechanical keyboards to come with the Mac media keys layout on the function row, allowing Mac users to enjoy the same layout they would find on traditional Mac computers.

The keyboard also comes with 3 OS-specific keycaps for Windows and Mac, respectively.

Keys and Layout

This compact keyboard still has all the important keys needed for productivity such as Page Up/Down, Home, End, and Screen Capture.

The N-key rollover feature allows the user to press as many keys at once as possible without losing any characters.

  • Switches: available options are

  • Backlight:

    • RGB backlight

  • Frame
    • Aluminium
    • Plastic

Keyboard shortcuts

Bluetooth: fn + (1/2/3) for up to 3 devices

Backlight color (RGB models): fn + < or >

Backlight effect (RGB or white backlight models): backlight key

Backlight On/Off: fn + backlight key

The keyboard has a built-in auto-sleep mode. To turn it off: fn + S + O hold for 4 seconds

To switch between function and multimedia row (F1 - F12): fn + X + L hold for 4 seconds

Factory reset: fn + J + Z hold for 4 seconds
